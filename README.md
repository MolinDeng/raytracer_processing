## Features

- Phong shading
- BVH acceleration structure
- Instancing
- Self-defined scene file
- Reflection with Recursive Ray Tracing
- Glossy Material
- Depth of Field
- Anti-aliasing with jittered sampling
- Motion blur
- Area light
- Soft shadow
